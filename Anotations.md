# **Anotações** :memo:

**Reverter a ultima migration executada**

```js
yarn typeorm migration:revert
```

**Criar entidade automaticamente**

```js
yarn typeorm entity:create -n User
```

**Biblioteca para manipular dados de entidades**

```js
yarn add class-transformer
```
